# 倪海厦中医体系 🏥

基于倪海厦经方体系的中医诊断与查询工具，整合《伤寒论》《金匮要略》《黄帝内经》《神农本草经》经典原文，结合六经辨证、八纲辩证、脉象、方药、针灸穴位速查。

## 功能模块

| 模块 | 内容 |
|------|------|
| 🔍 诊断助手 | 症状采集 → 六经辨证 → 方剂推理，29条诊断规则，51个症状（白话标注） |
| 🫀 脉象查询 | 28种脉象分类速查（浮沉迟数虚实），含脉形、主病、详解 |
| 💊 方剂速查 | 68首经方，含组成、剂量、用法、禁忌、倪师注解 |
| 🌿 药物手册 | 53味常用中药，性味归经、功效主治、倪师点评 |
| 📍 针灸穴位 | 十二经脉 + 奇经八脉，穴位定位含寸→厘米换算 |
| 🔗 经络循行 | 十二经脉完整循行路线 |
| 📕 伤寒论 | 117条原文，分七篇（太阳上/下、阳明、少阳、太阴、少阴、厥阴） |
| 📗 金匮要略 | 15篇原文（脏腑经络、痉湿暍、百合病、胸痹、痰饮等） |
| 📘 黄帝内经 | 6篇经典（上古天真论、四气调神大论、生气通天论等） |
| 📙 神农本草经 | 50味药物（上中下三品，按玉石/草/木分类） |
| ⚖ 八纲辩证 | 阴阳·表里·寒热·虚实体系 |
| ☯ 六经辨证 | 太阳·阳明·少阳·太阴·少阴·厥阴框架 |

## 技术栈

- 纯前端 HTML/CSS/JS，无需框架
- Capacitor 跨平台打包（Android + iOS）
- Nginx Docker 部署（群晖 NAS）
- GitHub Actions 自动构建

## 快速开始

```bash
# 本地开发
cd app
python3 -m http.server 8765 --bind 127.0.0.1
# 打开 http://127.0.0.1:8765
```

## Docker 部署（群晖）

```bash
docker-compose up -d --build
# 访问 http://<nas-ip>:8765
```

## 移动端

- **Android**：下载 [Actions](https://github.com/hotterry/tcm-diagnosis/actions) 中的 `android-apk` 产物
- **iOS**：克隆仓库后 `npx cap open ios` 在 Xcode 中构建签名
